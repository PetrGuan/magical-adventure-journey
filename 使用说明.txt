神奇的探险之旅 · 课堂使用说明
================================

作者：管Sir（GitHub: PetrGuan）
适用：人教版小学语文五年级下册第六单元习作《神奇的探险之旅》

——————————————————————————————

【运行方法（推荐）】

1) 把这个压缩包整体解压到任意位置（桌面也行）。
2) 进入解压后的文件夹。
3) 双击  start.bat
4) 黑色命令窗口会出现，2 秒后浏览器自动打开。
5) 等加载条走完 → 点「进入营地」。

⚠️ 不要直接双击 index.html，那是简化版（看不到大本营画面、人物会变成占位图）。
⚠️ 必须保留 start.bat 旁边的所有文件夹（src / vendor / assets / dist），缺一不可。

——————————————————————————————

【课堂操作】

进入大本营后：
  • 鼠标按住拖动 = 转视角看四周
  • 点击任一探险家 = 听他自我介绍（带视频）
  • 点击场景中央的「帐篷」标记 = 弹出装备清单
  • 点击远方的「去哪儿探险？」金色标记 = 弹出 5 个目的地选择卡
      └── 再点其中一个目的地 = 全屏播放该场景的 2 秒动图
  • 按 ESC 关闭弹窗
  • 关闭浏览器即结束

教室建议：
  • 按 F11 全屏，画面更沉浸
  • 教室音箱外放，全班一起体验
  • 整个体验控制在 3-5 分钟，给后续写作留时间

——————————————————————————————

【浏览器要求】

  ✅ Chrome / Edge / 360 极速模式（推荐）
  ❌ Internet Explorer（不支持）

——————————————————————————————

【遇到问题】

▶ 双击 start.bat 后什么都没反应
   → 可能是杀毒软件拦截了 PowerShell。右键 start.bat → 以管理员身份运行。
   → 或者打开浏览器手动访问  http://localhost:8000/index.html

▶ 浏览器打开后画面是空白
   → 等 5 秒再刷新一次（首次加载需要解码视频）。
   → 确认你用的是 Chrome 或 Edge。

▶ 听不到背景音乐 / 角色配音
   → 确认电脑没静音、浏览器音量没关。
   → 浏览器有时拦截自动播放：先点一下「进入营地」按钮即可解锁。

▶ 想关闭程序
   → 直接关浏览器窗口；同时关掉那个黑色命令窗口（×）即彻底结束。

——————————————————————————————

【发现问题，需要开发者远程定位】

如果出现「画面不正常 / NPC 不出现 / 听不到声音」等任何异常：

1. 启动 start.bat 后浏览器自动开页面，**不要关掉**。
2. 在地址栏的网址末尾**手动加上**  ?debug=1  （问号后面跟 debug 等于 1）
   完整地址例：  http://localhost:8000/index.html?debug=1
3. 按回车刷新。
4. 屏幕**左下角**会出现一块黑色调试日志面板，里面记录了所有错误。
5. 等加载完之后，点面板顶部的 **📋 复制全部** 按钮。
6. 把复制到的内容**粘贴到聊天软件发回开发者**，问题基本能定位。

如果连黑色命令窗口都闪退看不到，截一张当时的桌面/弹窗发回也行。
