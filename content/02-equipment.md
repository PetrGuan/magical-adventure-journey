# 装备 · 大本营装备桌（纯装饰）

> 装备桌是大本营场景里的**装饰元素**——纯展示，**不需要点击交互**。
> 五年级学生一眼就能认出这些是探险家会带的东西。
> 老师可以指着说："这就是探险家要准备的装备，你想带哪些？"
> 让学生**自己想象**怎么用——这是写作素材的来源。

---

## 桌上摆着这些东西

| 物品 | 老师讲解备用 |
|---|---|
| 🧭 指南针 | 辨方向 |
| 🗺️ 地图（卷起来） | 规划路线 |
| 🔭 望远镜 | 看远处 |
| 💧 水壶 × 4 | 储水（探险一天 2 升） |
| 🍫 压缩饼干 / 巧克力 | 干粮 |
| 💊 急救药箱（红十字标志） | 救命用 |
| ⛺ 卷起的帐篷 | 过夜 |
| 🪢 登山绳（一卷） | 攀爬 / 救援 |
| 🔦 手电筒 | 照明 |
| 🎒 大背包 | 装东西 |

---

## 3D 场景里的实现

- **桌子 × 1**：木质长方形（Sketchfab 关键词 "wooden table low poly"）
- **物品 GLB 模型**：散落桌面，**不要太整齐**——像真的探险家正在收拾装备
- **桌面附近灯光稍亮**：上方一盏聚光灯，让装备更显眼
- **互动反馈**：学生鼠标 hover 装备时，物品**轻微浮起 + 发淡光**——但**不弹窗**，不打断节奏

**为什么不做点击说明？**
- 学生看 6 个 NPC 自我介绍已经吸收了大量信息
- 装备说明会让节奏太满
- 让"为什么带这个"成为**写作时的思考点**，而不是被动接收的知识
