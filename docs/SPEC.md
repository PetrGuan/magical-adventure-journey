# 神奇的探险之旅 · 项目需求规格 (SPEC)

> 本文档是项目的**需求圣经**。所有功能 / 设计 / 实现决策必须可追溯到这里。
> 所有 issues 必须引用本文档的具体章节作为验收依据。

## 1. 项目概述

| 字段 | 内容 |
|---|---|
| **客户** | 小学五年级语文老师 |
| **终端用户** | 五年级学生（10-11 岁） |
| **上下文** | 人教版小学语文五年级下册第六单元习作课 |
| **课时使用** | 一节 40 分钟作文课的**开头 3-5 分钟** |

### 1.1 核心目标

在《神奇的探险之旅》习作课的**开头 3-5 分钟**，
通过 360° 沉浸式探险大本营场景，
让学生在动笔前"进入"探险家身份，激发想象力，
从而写出更生动的作文。

### 1.2 不在范围内（明确否定）

- ❌ 完整探险游戏（**这不是游戏**）
- ❌ 多场景漫游（只有大本营**一个**场景）
- ❌ 强制学生做选择（"挑选伙伴 / 装备 / 目的地"是**心理过程**，不是 UI 流程）
- ❌ 多人同时使用（一个老师一个大屏，全班一起看）
- ❌ 需要登录或保存

## 2. 课本原题（习作题目原文）

### 2.1 题目导言

> 你喜欢探险吗？你读过有关探险的书吗？
> 这次习作就让我们编一个惊险刺激的探险故事吧。

### 2.2 人物（学生从两列各选一人组队）

| A 列：成年导师 | B 列：同龄伙伴 |
|---|---|
| 经验丰富的探险爱好者 | 好奇心强、性格活泼的妹妹 |
| 知识渊博的生物学家 | 胆子大但行事鲁莽的表哥 |
| 见多识广的向导 | 心细而胆小的同学 |

### 2.3 场景（学生选 1 个）

茫茫大漠 / 热带雨林 / 海中荒岛 / 幽深洞穴 / 南极冰川 ……

### 2.4 装备（参考清单）

指南针、地图、饮用水、食品、药品、探照灯、瑞士军刀 ……
（帐篷在工具里**不是装备**——它是装备卡片的入口热点。）

### 2.5 险情（参考清单）

遭遇猛兽、暴雨来袭、突发疾病、断水断粮、落石雪崩 ……

### 2.6 写作指导

> 写的时候要展开丰富合理的想象，
> 把遇到的困境、求生的方法写具体，
> 如果能把心情的变化写出来就更好了。
> 写完后，认真修改自己的习作。

### 2.7 课后延伸

> 如果有兴趣，可以续编你的探险故事。

## 3. 老师需求（产品功能）

### 3.1 场景设定（核心叙事）

> "马上要出发探险了。在出发之前，我们在一个像大本营的地方。"

### 3.2 学生在场景中能做什么

#### 3.2.1 挑选伙伴（心理过程）

> "你可以挑选你的伙伴，就是和谁一起去探险组队。"

**实现要求**：6 位 NPC 在场景中可见且可点击 → 弹出**角色卡片**听自我介绍。
**重要**：**不需要勾选式 UI**——挑选是学生在心里决定的事。
角色卡片**只**承载「认识每一个人」，**不混入装备、不混入目的地**。

#### 3.2.2 挑选装备（心理过程）

> "也可以挑选就是带上一些装备。"

**实现要求**：场景里有一个**帐篷热点**（陈老师与王叔叔之间地面），点击后弹出**装备卡片**：
- 顶部旁白："你在帐篷里发现了很多探险装备：指南针、地图、饮用水、食品、药品、探照灯、瑞士军刀…… 准备带上哪些呢？"
- 主体一张帐篷内装备合成图（纯展示，不勾选）
**为什么文字+图**：客户希望有「单机游戏初始阶段拣选装备」的氛围，但仍维持「挑选是心理过程」原则。

#### 3.2.3 选择目的地（心理过程）

> "还可以选择你想去什么目的地。"

**实现要求**：场景里有一个**远方树/标记物热点**（左前方），点击后弹出**目的地选择卡**：
- 顶部："你想去哪里探险呢？"
- 5 个目的地按钮（沙漠 / 雨林 / 荒岛 / 洞穴 / 冰川）
- 点击任一按钮 → 全屏播放该目的地的 2 秒循环 mp4（VideoModal）
**重要**：5 个按钮是**预览入口**，不是单选 UI——学生看完心里决定。

#### 3.2.4 三个交互彼此独立

工具把「选伙伴 / 选装备 / 选目的地」三件事拆成场景里的**三个独立热点**，
对应课堂叙事的三步：①认识同伴 → ②查看装备 → ③望向远方决定去哪。
卡片之间互斥显示，不叠层。

### 3.3 唯一场景

> "只需要呈现一个场景，就是大本营。"

### 3.4 6 位 NPC 在场景中

> "大本营里面有 6 个人站在这里。
>  我点到每个人，然后这个人就开始自我介绍说话。"

**6 个人**：见 §2.2，对应课本两列人物。
**自我介绍**：用 TTS 中文配音，每段 15-20 秒（详见 §5.2）。

### 3.5 装备入口（帐篷热点）

> [v2 客户调整] "选择装备的话就是点击那个帐篷……
>  弹出来一个交互的卡片，"你在帐篷里发现了这些装备……你想带上哪些呢？""

**设计含义**：
- 场景里有一顶帐篷的可点击标记（陈老师与王叔叔之间）
- 点击帐篷弹出**装备卡片**（见 §3.2.2）
- 装备本身是一张图——视觉识别度优先（一眼看出是什么），**不要文字介绍**
- **挑选仍是心理过程**——卡片里的装备图不需要勾选/选中态

### 3.6 目的地预览（远方树热点）

> [v2 客户调整] "点击远方的一棵树或者场景里任何一个标记，
>  弹出来一个卡片，"你想去哪里探险呢"，下面是几个目的地，
>  点开比如茫茫沙漠，那个沙漠就出来了。"

**实现**：
- 场景里有一棵远方的树（或任意标记物）做点击热点
- 点击 → 弹出目的地选择卡（见 §3.2.3）
- 选择目的地 → 全屏播放 2 秒循环 mp4（VideoModal）

**注**：v1 曾把 5 张挂在墙上做画框；v2 已改为通过卡片展开，画面不再被画框占据。

### 3.7 课堂使用场景

> "这是一节课的开头，相当于给他们创设一个情景。
>  因为这节课是要写作文呀，写一篇那个想象作文去探险，
>  所以一开始让他们进入到一个探险的那种，一个真实的情境当中吧。"

## 4. 课堂使用流程

### 4.1 时长

整体体验：**3-5 分钟**

### 4.2 在 40 分钟课时里的位置

| 时间 | 环节 |
|---|---|
| 0:00 – 1:00 | 老师导入 |
| 1:00 – 5:00 | 学生体验大本营（**本工具的工作时段**） |
| 5:00 – 7:00 | 老师收回 + 写作要求讲解 |
| 7:00 – 35:00 | 学生写作（28 分钟） |
| 35:00 – 40:00 | 展示交流 |

### 4.3 设备环境

| 项 | 配置 |
|---|---|
| 主机 | 教室 PC（Windows + Chrome / Edge） |
| 屏幕 | 投影或大屏（1080p / 4K） |
| 音频 | 教室音箱外放（不戴耳机） |
| 显示 | F11 全屏 |
| 操作 | 学生上台用鼠标，全班看大屏 |

## 5. 关键技术决策

### 5.1 架构：360° 全景图 + sprite 热点

**为什么不用实时 3D**：
实时低多边形 3D 渲染永远到不了 Nibiru 那种"专业渲染 360° 全景"画质。
同样架构（全景图 + 热点）我们用 AI 工具也能 30 秒生成一张。

**为什么不需要走动**：
3-5 分钟课堂引入里，学生原地转视角看一圈就够。
WASD 操作五年级不熟悉，反而浪费宝贵时间。

### 5.2 NPC 自我介绍：中文 TTS 配音

**Edge TTS** 免费、高质量、6 种音色满足不同年龄/性别。
每段控制在 **15-20 秒**——五年级注意力 + 课堂时间紧。

| 角色 | 音色 ID | 风格 |
|---|---|---|
| 李大山 | zh-CN-YunjianNeural | 沉稳低音、中年男 |
| 陈雨桐 | zh-CN-XiaoxiaoNeural | 温柔知性、青年女 |
| 阿木叔 | zh-CN-YunfengNeural | 豪爽、中年男 |
| 小雨 | zh-CN-XiaoshuangNeural | 活泼、女童 |
| 张大壮 | zh-CN-YunxiaNeural（高音变速） | 大嗓门、男少年 |
| 小明 | zh-CN-YunxiaNeural（低音变速） | 腼腆、男童 |

详见 [`design/tts-voice-guide.md`](../design/tts-voice-guide.md)。

### 5.3 目的地预览：AI 视频生成

**即梦 / 可灵** 中文友好，免费额度足够。
2 秒循环 mp4，每个 < 5 MB。

详见 [`content/04-destinations.md`](../content/04-destinations.md)。

### 5.4 内容尺度（针对 10-11 岁）

- ✅ 紧张 + 探险氛围
- ✅ 险情都有可解决方案，传递「勇敢 + 智慧 = 化险为夷」
- ✅ 强调好奇心、互助、成长
- ❌ 恐怖、悬疑、死亡暗示
- ❌ 复杂选择 UI

### 5.5 交付形式

单文件夹，**浏览器双击 `start.command` / `start.bat` 即用**。
**离线运行**（生成阶段需联网，运行时不需要）。

## 6. 验收标准

### 6.1 内容验收
- [ ] 6 位 NPC 名字、性格、口头禅 = 课本两列人物的具象化（§2.2 → §3.4）
- [ ] 5 个目的地 = 课本场景列表（§2.3）
- [ ] 装备桌物品 = 课本装备列表（§2.4）
- [ ] 内容无恐怖、无死亡暗示、五年级友好（§5.4）

### 6.2 技术验收
- [ ] 双击 `start.command` 或 `start.bat` 能启动
- [ ] Chrome / Edge 主流浏览器都能跑
- [ ] 加载完成 < 10 秒
- [ ] 全屏后画面无变形
- [ ] 教室音箱能听到 NPC 配音和背景音

### 6.3 课堂验收
- [ ] 学生 0 学习成本进入互动（§4.3）
- [ ] 整体体验时长落在 3-5 分钟（§4.1）
- [ ] 老师按 ESC 能立即退出
- [ ] 课后学生作文里的角色 / 场景 / 险情描写比往届更具体
